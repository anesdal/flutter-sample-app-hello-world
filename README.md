# IPTV Player Pro 📺

تطبيق IPTV متكامل مبني باستخدام Flutter يدعم:
- نظام **Xtream Codes API**.
- قوائم **M3U** (روابط وملفات).
- مشغل داخلي (Better Player) ومشغلات خارجية (MX Player).
- دعم شعارات القنوات بنظام الـ Caching.

## طريقة البناء (Build)
1. قم بعمل `Fork` للمستودع.
2. اذهب إلى تبويب `Actions`.
3. اختر `Build Android APK` واضغط `Run workflow`.
